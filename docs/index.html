04快速开始

# 安装 wx-cli
npm install -g @jackwener/wx-cli

# 签名微信（macOS，仅首次）
codesign --force --deep --sign - /Applications/WeChat.app
killall WeChat && open /Applications/WeChat.app
sudo wx init

# 启动分析器
cd wechat-analyzer
pip install flask requests
python3 server.py
# → http://localhost:8899
隐私优先 — 所有计算在本地完成，数据不出设备。API Key、昵称等配置保存在项目 config.json 中，与 Skill Hub 发布的技能包完全分离，用户隐私不受影响。
